# gadgethub
🛒 GadgetHub - Electronics E-Commerce Web Application
📖 Overview

GadgetHub is a full-stack e-commerce web application for buying and selling electronic gadgets. It provides separate modules for Users and Administrators, allowing customers to browse and purchase products while enabling administrators to efficiently manage the entire store.

The application is developed using Java Servlets, JSP, HTML, CSS, Bootstrap, Oracle Database, and JavaMail API for sending email notifications.

⚙️ Installation
Prerequisites
Java JDK 8 or later
Apache Tomcat 9+
Oracle Database
Eclipse IDE or IntelliJ IDEA
Steps
Clone the repository.
git clone https://github.com/your-username/GadgetHub.git
Import the project into Eclipse or IntelliJ IDEA.
Configure the Oracle Database connection.
Execute the SQL scripts to create the database tables.
Add the Oracle JDBC Driver to the project.
Configure JavaMail API credentials.
Deploy the project on Apache Tomcat.
Start the Tomcat server.
Open the application:
http://localhost:8080/GadgetHub
